# RunRunRun!
シンプルでスタイリッシュな2Dランニングアクションゲーム。プレイヤーを操作して迫りくる敵や障害物を避け、撃ち、踏みつけ、ハイスコアを目指そう！

## 〇ゲーム概要
プレイヤーは白いキャラクターを操作し、右から流れてくるエネミーやオブジェクトを攻略しながらひたすら走る。時間が経つにつれて移動速度が上がり、難易度が上昇。
## 〇ルール
### 1.スコア
- 走っている間、スコアが自動的に加算
- 敵を弾で倒すと+50点、踏みつけて倒すと+100点
### 2.敵と障害物
- いずれもぶつかるとゲームオーバー
- エネミーは球や踏み付けで倒せるが、オブジェクトは壊せない（上に乗ることは可能）
### 3.難易度
- スコアが1000点貯まるごとに、ゲームのスクロールスピードが徐々にアップ
## 〇機本操作
- w/↑：ジャンプ/二段ジャンプ
- SPACE：弾を打つ
- R：ゲームオーバー時リスタート
## 〇機能
- 二段ジャンプ：空中でさらにもう一度ジャンプが可能
- 攻撃：弾を打って敵を倒すことが可能
- 踏みつけ：敵を上から踏むことで倒し、さらに高く跳ね上がることが可能
- 障害物（オブジェクト）：撃破不可能な障害物は、上に乗ることで足場として利用可能
## 〇追加予定の機能
### 1.アイテム
* 連射
* パルス
* 一回ガード
* ロケット（一定距離進む）
* 助っ人
### 2.システム
* HP
* ボス
* enemyの種類
* 障害物の種類
* ステージの種類
* スコア履歴
## 〇使用技術
- Python 3.x
- Pygameライブラリ
## 〇環境構築
### 1.必要ライブラリのインストール
pip install pygame
### 2.ゲームの起動：ソースコードがあるディレクトリで以下のコマンドを実行
python main.py
### 3.コード構造
- player：プレイヤーの物理挙動と描画を管理
- Enemy: 敵キャラクター。左へ移動し、画面外に出ると自動削除される
- Obstacle: 破壊不能な障害物
- Bullet: プレイヤーが発射する弾丸
- game_loop(): ゲームのメインループ（イベント、更新、衝突判定、描画）
